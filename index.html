# What is Bootstrap Confirm ? The Bootstrap Confirm popup is set of reusable element made of twitter bootstap & jquery. Whenever you found responsive & reusable javascript confirm, you can use this plugin. It has all you need for any confirmation action like title , content , callback events etc.

# How to use ? $(":submit").BootConfirm();
